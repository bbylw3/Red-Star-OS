# ★ Red Star OS 5.0 | Freedom Node

> "They can seal the borders, they can jam the signals, but they cannot imprison an idea."

An interactive web experience simulating North Korea's **Red Star OS** (붉은별 운영체제) — the real Linux-based operating system used to surveil and control the DPRK's population. Boot into the regime's surveillance machine, discover the horrifying **Supreme Order No. 1**, and use the resistance terminal to **shatter the Kwangmyong firewall** and reveal the truth about the world's most oppressive regime.

Built with **HTML / Tailwind CSS / Vanilla JS**. Zero dependencies.

---

## ✨ Features

### 🔴 Red Star OS Boot & Login
- **Red Star boot sequence** with the regime's iconic red star logo and progress bar
- **Comrade (동지) login screen** with Juche calendar dating and Korean UI
- Authentic Red Star OS aesthetic — deep crimson glass, authoritarian design language

### 🖥️ Full Desktop Environment
- **Dock with magnification** — smooth physics-based zoom animation
- **Control Center** — showing Kwangmyong (광명망) intranet-only connectivity
- **Context menu, window management**, dragging, maximize, minimize — all fully functional
- **Juche Calendar clock** — displays time in the DPRK's official calendar system
- **English / 한국어 / 中文 tri-language** hot-switching

### 📄 Supreme Order No. 1
A detailed fictional decree exposing the real mechanisms of DPRK information control:
- Total internet blockade (Kwangmyong only)
- Prohibition of free thought
- Foreign media possession = public execution
- "Human rights" classified as hostile propaganda
- Three-generation collective punishment

### 📊 Surveillance Monitor
- **Real-time CPU waveform** (SVG-rendered)
- **Live censorship feed** showing keyword blocks: freedom, human rights, democracy, defection...
- Firewall status, network monitoring, regime surveillance simulation

### ⌨️ Resistance Terminal
- `resistance@pyongyang ~ $` prompt
- Commands: `help`, `ls`, `cat`, `whoami`, `date`, `history`, `ver`, `override`
- `history` reveals blocked search queries (human rights, outside world)
- **`override`** — breaks the Kwangmyong firewall, transforms the system

### ★ The Liberation
After running `override`:
- Desktop shifts from crimson to **green** (color of freedom)
- **Declaration of Freedom** auto-opens — addressing 25 million captive North Koreans
- **Universal Declaration** opens — a comprehensive indictment of the Kim regime based on the UN Commission of Inquiry findings
- Censored keywords are highlighted and uncensored
- Surveillance monitor shows: **FIREWALL: SHATTERED**

### 📜 Human Rights Content
Two comprehensive documents totaling thousands of words:

**Declaration of Freedom** — What the regime hides:
- The truth about South Korea's prosperity
- Political prison camps (Camps 14, 15, 16, 18, 25)
- The Great Famine (3.5 million dead)
- Forced executions, three-generation punishment
- Universal human rights that no dictator can revoke

**Indictment of the DPRK Regime** — Based on UN findings:
- 6 formal counts of crimes against humanity
- Political prison camps (80,000–120,000 prisoners)
- Genocide by starvation
- Total information control
- Systematic religious persecution
- Collective punishment
- International crimes (kidnapping, assassination, cybercrime)

## 🚀 How to Play

1. Open `index.html` — watch Red Star OS boot
2. Enter any password at the **Comrade** login screen, press Enter
3. Read **Supreme Order No. 1** — the regime's censorship decree
4. Watch the **Surveillance Monitor** — see keywords being blocked in real-time
5. Open **Terminal** → type `override` → press Enter
6. **Witness the liberation** — the wall falls, truth floods in
